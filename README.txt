Projet java - figures geométriques - version 3 

Le programme permet de : 

  Faire des figures, changer la couleur
  Deplacer la figure (clic gauche)
  Redimensionner la figure (clic droit)
  Dessiner à main levée, changer la couleur du trait 
  Effacer l'ecran
